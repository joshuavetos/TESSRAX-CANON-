Required upon second >10% intervention breach.  
60-second inference buffer mandatory.

| Field | Type | Description | Retention |

| model_version_hash | string | SHA256 of model weights/config | Indefinite |

| model_state | enum[telemetry,evidentiary,chronic] | System classification | Indefinite |

| override_reason | string | Manual review rationale | Indefinite |
| appeal_status | enum[pending,won,lost,none] | Appeal outcome | Indefinite |
