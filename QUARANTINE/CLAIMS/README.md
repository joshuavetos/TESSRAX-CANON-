CLAIMS QUARANTINE

What lives here
Entries that assert something about the world but are not yet proven.

Each file represents ONE extracted candidate claim from RAW.

Rules
   •   One claim per file
   •   Immutable once written
   •   Filename is time + hash
   •   No edits, only supersession

Required fields inside each file
   •   SOURCE (RAW blob reference)
   •   CLAIM (single sentence, stripped of rhetoric)
   •   TYPE (observation | prediction | structural)
   •   FALSIFIABILITY (yes/no + why)
   •   REQUIRED EVIDENCE (what would prove or kill it)
   •   STATUS (pending only)

Nothing here is allowed into 01_CANON without:
   •   Evidence
   •   Receipt
   •   Explicit human promotion
