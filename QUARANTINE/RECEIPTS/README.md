RECEIPTS

What lives here
Proof that something was checked, tested, verified, or falsified.

Examples
   •   Hash verification logs
   •   CI run outputs
   •   Tool execution logs
   •   External confirmations
   •   Screenshots of reality

Rules
   •   Must be machine-generated or human-signed
   •   Must reference a Claim ID
   •   Immutable
   •   Required for any CLAIM → CANON promotion

No receipt = no truth.
